# Smart City Project

The Smart City project is a web-based platform that enables citizens to report city-related issues such as garbage management, road damage, water supply problems, and other public service complaints. 

The system helps local authorities monitor and resolve issues efficiently while improving communication between citizens and the administration.

This project is developed collaboratively using Git and GitHub.

## Features

- Citizen complaint registration
- Issue tracking system
- Admin dashboard for complaint management
- Real-time status updates
- User-friendly web interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Git & GitHub
